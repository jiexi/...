# Arch Linux Setup Scripts

### Download:

```shell
curl -4 https:/raw.githubusercontent.com/jiexi/.../master/archlinux/setup/init -o init
curl -4 https:/raw.githubusercontent.com/jiexi/.../master/archlinux/setup/chroot -o chroot
```

### Verify:

```shell
sha1sum init chroot
```

### Install:

```shell
bash init
```
