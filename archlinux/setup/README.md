# Arch Linux Setup Scripts

### Download:

```shell
curl -4 -O https:/raw.githubusercontent.com/jiexi/.../master/archlinux/setup/init
curl -4 -O https:/raw.githubusercontent.com/jiexi/.../master/archlinux/setup/chroot
```

```shell
sha1sum init chroot
```

```shell
bash init
```
